# Desktop Pet

Desktop Pet 是一个 macOS 桌面宠物应用，当前内置「吉伊」桌宠。它把透明悬浮桌宠、可爱的互动陪伴和轻量专注计时放在同一个小小窗口里：吉伊会待机、害羞、挥手、奔跑、学习，也可以陪你开始专注或休息。项目已经具备多桌宠配置化基础，后续可以继续扩展更多角色。

## 功能总览

### 桌宠基础能力

- 透明无边框悬浮窗口
- 始终置顶，并可显示在所有工作区
- 鼠标拖动桌宠移动窗口
- 自动记住窗口位置，下次启动恢复
- 支持登录时自动启动
- macOS Finder / Dock / Tray 使用统一品牌图标

### 吉伊互动能力

- `idle` 待机动画
- 鼠标悬停触发害羞状态
- 单击挥手，并弹出随机气泡台词
- 双击进入 / 退出学习模式
- 拖动时根据方向播放左右奔跑动画
- 长按触发惊讶 / 弹一下动画
- 待机时自动冒出气泡台词

### 专注陪伴功能

- 右键吉伊打开 React 渲染的「吉伊专注助手」面板
- 预设 25 / 45 分钟专注，以及 5 / 10 分钟休息
- 支持自定义专注或休息计时
- 桌宠上方保留倒计时标签
- 支持暂停、继续、结束当前计时
- 支持自动进入下一阶段
- 记录今日专注与休息统计
- 专注设置支持：
  - 默认专注时长
  - 默认短休息时长
  - 长休息开关、触发轮次与时长
- 专注状态会与吉伊的 `study` 动画联动

### 应用级能力

- 菜单栏 Tray 控制
- 显示 / 隐藏桌宠
- 切换当前桌宠
- 登录时自动启动开关
- 退出应用

### 架构能力

- `PetConfig` 多桌宠配置化基础
- 当前默认宠物为吉伊
- 宠物动画、帧尺寸、缩放、台词等能力集中在宠物配置中
- 后续可以扩展更多桌宠并接入切换能力

## 运行方式

安装依赖：

```bash
npm install
```

启动开发模式：

```bash
npm run dev
```

类型检查：

```bash
npm run typecheck
```

构建：

```bash
npm run build
```

## macOS 打包

打包 `.app`：

```bash
npm run package:mac
```

打包完成后，应用大致输出在：

```text
out/Desktop Pet-darwin-arm64/Desktop Pet.app
```

可以在 Finder 中双击 `Desktop Pet.app` 运行。

## 使用说明

### 基础交互

- 单击吉伊：挥手并显示随机气泡台词
- 双击吉伊：进入或退出学习模式
- 拖动吉伊：移动桌宠窗口，拖动时播放奔跑动画
- 长按吉伊：触发惊讶 / 弹一下动画
- 悬停吉伊：触发害羞状态
- 右键吉伊：打开「吉伊专注助手」面板

### 专注面板

- 右键吉伊打开面板，再次右键面板、点击关闭按钮，或点击面板外部可关闭
- 使用快捷按钮开始 25 / 45 分钟专注，或 5 / 10 分钟休息
- 使用「自定义计时...」输入 1 到 180 分钟之间的专注或休息时长
- 使用暂停 / 继续 / 结束控制当前计时
- 打开「自动进入下一阶段」后，专注完成会进入休息，休息完成会进入下一轮专注
- 在「专注设置...」中调整默认专注时长、默认短休息时长、长休息规则
- 今日统计会显示当天完成的专注次数、专注分钟数、休息次数与休息分钟数

### Tray 菜单

Tray 菜单保留应用级控制，避免把专注计时入口继续堆到系统菜单中：

- 显示 / 隐藏桌宠
- 切换桌宠
- 登录时自动启动
- 退出应用

专注计时的主要入口是右键吉伊打开的 React 面板。

## 项目结构

```text
src/main
```

Electron 主进程。负责创建透明悬浮窗口、Tray 菜单、登录项、窗口位置持久化、通知、本地数据读写，以及 main/renderer 之间的 IPC 处理。

```text
src/preload
```

Electron preload 层。通过 `contextBridge` 暴露安全的 `window.desktopPet` API，连接 renderer 与主进程能力。

```text
src/renderer
```

React 渲染层。负责桌宠画面、动画状态、点击/双击/拖动/长按、气泡系统、专注面板、倒计时标签与专注计时交互。

```text
src/renderer/pets
```

桌宠配置层。当前包含吉伊的 `PetConfig`，描述 spritesheet、帧尺寸、动画状态、缩放和台词等。

```text
src/shared
```

主进程和渲染层共享的类型与纯逻辑，例如专注计时状态、默认设置、宠物注册表。

```text
src/types
```

全局类型声明，主要用于声明 renderer 中可访问的 `window.desktopPet` API。

## 数据持久化

应用会把本地状态保存到 Electron `userData` 目录下。当前使用的文件包括：

- `window-state.json`：桌宠窗口位置
- `pet-preferences.json`：当前选中的桌宠
- `focus-timer.json`：当前专注 / 休息计时状态
- `focus-stats.json`：今日专注与休息统计
- `focus-preferences.json`：专注设置，包括自动进入下一阶段、默认时长和长休息规则

这些数据只保存在本机，不会提交到仓库。
